# Nebula Strike

A fast-paced space shooter built with Vanilla JavaScript and HTML5 Canvas.

## How to Run
1. Ensure you have a web browser installed.
2. Unzip the files or keep them in a folder structure matching the filenames.
3. Open `index.html` in your browser.

## Controls
- **Move Mouse**: Control your ship.
- **Left Click**: Shoot bullets.
- **Spacebar**: Unleash (only when meter is full).
- **Escape**: Pause game.
- **Click (Menu/Game Over)**: Restart game.

## Features
- 4-layer parallax background.
- Particle system for explosions and trails.
- Tier-based weapon upgrades.
- Boss fights and powerups.
- Screen shake effects.
